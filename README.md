# Dice Game

A two-player dice rolling game built with vanilla JavaScript and DOM manipulation.

## About
Two players each roll a dice simultaneously. The player with the higher number wins that round. Built as a hands-on exercise in DOM manipulation and dynamic content updating.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Play
1. Open the game in your browser
2. Refresh the page to roll the dice
3. The player with the higher number wins

## How to Run
Simply open `index.html` in any browser — no installation needed.
